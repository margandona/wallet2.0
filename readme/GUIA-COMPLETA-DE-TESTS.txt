CONCEPTOS CLAVE DE MOCKITO:

1. **mock()** - Crea un objeto simulado
   ```java
   IUsuarioRepository repo = mock(IUsuarioRepository.class);
   ```

2. **when().thenReturn()** - Define comportamiento
   ```java
   when(repo.existePorEmail(any())).thenReturn(false);
   ```

3. **verify()** - Verifica que se llamÃ³ un mÃ©todo
   ```java
   verify(repo).guardar(any(Usuario.class));
   ```

4. **never()** - Verifica que NO se llamÃ³
   ```java
   verify(repo, never()).guardar(any());
   ```

5. **any()** - Acepta cualquier argumento
   ```java
   any(Email.class)
   any(Usuario.class)
   ```
